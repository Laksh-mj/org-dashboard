# OrgOS — Enterprise Organization Dashboard

A fully featured, single-file enterprise dashboard built with HTML, CSS and JavaScript.

## 🔗 Live Demo
👉 https://laksh-mj.github.io/org-dashboard/

## ✨ Features
- 🏢 Multi-Organization Support
- 📦 Inventory Management with low-stock alerts
- 🛒 Order Management with GST calculation
- 🚚 Delivery Tracking
- 🧾 Billing & Invoice Generation (GST-ready, UPI/Card/Cash)
- 📊 Analytics & KPI Dashboard
- 💰 Finance & P&L Tracking
- 🔮 AI Forecast & Smart Suggestions
- 👤 Contacts & CRM with VIP segmentation
- 🏭 Supplier Management
- 📋 Full Audit Logs
- 🤖 AI Assistant Chatbot
- 🌙 Dark / Light Mode Toggle
- 💾 LocalStorage Data Persistence
- 📥 CSV Export for all modules
- 🔐 Role-Based Access (Admin / Manager / Staff)

## 🚀 Usage
Just open `index.html` in any browser — no server, no install needed.

## 🛠 Tech Stack
- Pure HTML5 + CSS3 + Vanilla JavaScript
- Chart.js for data visualization
- localStorage for data persistence
